# 安裝手冊

## 系統需求
- 作業系統：Windows, macOS, Linux
- Node.js 版本：14.x 或更高
- npm 版本：6.x 或更高

## 安裝步驟

1. 下載並安裝 [Node.js](https://nodejs.org/) 和 [npm](https://www.npmjs.com/)

2. 克隆專案到本地端：
    ```sh
    git clone https://github.com/your-repo/your-project.git
    ```

3. 進入專案目錄：
    ```sh
    cd your-project
    ```

4. 安裝相依套件：
    ```sh
    npm install
    ```

5. 建置專案：
    ```sh
    npm run build
    ```

6. 啟動應用程式：
    ```sh
    npm start
    ```
